## 🚀 Getting Started
To run Servlet, JSP, or JDBC examples:
1. Use an IDE like **Eclipse** with **Apache Tomcat 9.0**
2. Set up **MySQL** and update database credentials accordingly
3. Deploy the WAR or project to Tomcat server
